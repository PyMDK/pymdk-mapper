# pymdk-mapper
Applies Minecraft obfuscation mappings in [PyMDK format](https://github.com/PyMDK/PyMDK/blob/master/core/mdk-core/src/generation/mappings/mod.rs) to a jar file and emits the mapped jar file.

Usage: 
```
java -jar pymdk-mapper-1.0.0.jar <input-file> <mapping-file> <augmentations-file> <output-file>
```
